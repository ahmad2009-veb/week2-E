# week2-E
Обуна ба рассылка — корбар метавонад почтаи электронии худро ворид карда, ба хабарҳо ва аксияҳо обуна шавад.
Маълумоти тамос — рақами телефон, email ва вақти корӣ нишон дода шудааст.
Иконкаҳои шабакаҳои иҷтимоӣ — барои гузаштан ба Telegram, Instagram ва дигар шабакаҳо.
Маълумоти ширкат — адрес, ҳуқуқи муаллиф ва сиёсати махфият.

Усулҳои пардохт — Visa, Mastercard, Mir, Apple Pay ва Google Pay нишон дода шудаанд.
